# 1、数据中心
## 1.1、简介
   &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;将一些经常使用的数据统一管理起来，进行数据的维护，
   同时也方便其他应用进行数据的调用和访问。例如：省市区地区数据、ip地址数据、银行卡数据等。

## 1.2、构成
### 1.2.1、接口层
    数据中心接口层，定义接口实体和服务接口层，方便rpc、webservice调用
    省市区地区数据
    ip地址数据
    银行卡数据
    车商标、车型、车系数据
### 1.2.2、服务实现层
    数据中心服务层，主要是实现数据中心业务逻辑。
### 1.2.3、控制器层

    数据中心控制层，主要展示数据，并且维护数据。同时提供一些基本信息查询功能。
